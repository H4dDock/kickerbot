# kickerbot
The bot that help kicker players. Can store players, bets and game results.
Keeps custom bets.
## Commands:
1) Create new player: ```/c @username```
2) Create custom bet ```/b betname ["betDescription"] ```
3) Save game result ```/g @winner-@loser bettype count```
